Installation Instructions:

Extract the downloaded zip into your 7D2D Mods folder (create one if it does not exist).

To verify you have installed the mod correctly, the ModInfo.xml file should be located at Mods\ImprovedHordes\ModInfo.xml relative to your 7D2D install folder.
